## Cảm biến ánh sáng LDR

- LDR thay đổi điện trở theo ánh sáng:

  Sáng mạnh → điện trở LDR giảm

  Tối → điện trở LDR tăng
  
- Ta mắc mạch :  Vcc → LDR → ADC → R → GND

   Điện áp tại ADC = R / (R + LDR) × Vcc

   **Khi sáng** → LDR giảm → mẫu số (R+LDR) nhỏ → điện áp tăng → **ADC tăng**

- Ngoài ra còn cách mắc : Vcc → R → ADC → LDR → GND

  Khi sáng -> ADC giảm

- Thiết kế khi trời càng sáng thì LED càng tối , trời càng tối thì càng sáng
